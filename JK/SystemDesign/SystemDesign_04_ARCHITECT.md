# SystemDesign_04_ARCHITECT

## System Architecture Mastery

*Target Audience*: Staff Engineer / System Architect.  
*Philosophy*: Focus on "Design choices, Trade-offs, and Systems Theory".

### Internals

- Operating system kernels and system calls.  
- Network protocols and TCP/IP stack.  
- Database engine internals.

### Theory

- System theory and cybernetics.  
- Complexity theory in software systems.  
- Information theory in data transmission.

### Scalability

- Horizontal vs vertical scaling trade-offs.  
- Elasticity in cloud systems.  
- Global distribution strategies.

### Design

- Event-driven architectures.  
- Serverless computing patterns.  
- Edge computing designs.

### Patterns

- Saga pattern for distributed transactions.  
- Bulkhead pattern for fault isolation.  
- Sidecar pattern in microservices.
