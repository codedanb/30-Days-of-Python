# WebDevelopment_01_BASICS

## Introduction to Web Development

*Target Audience*: Absolute beginner (Grade school student / Non-technical person).  
*Philosophy*: Assume NO prior knowledge. Explain "what" and "why" before "how".

### What is the Web?

**Websites**  
Pages on the internet.

**Web Development**  
Building websites and web apps.

**Why Web Dev?**  
To create things people can use online.

### Basic Technologies

**HTML**  
Structure of web pages.

**CSS**  
Styling web pages.

**JavaScript**  
Making pages interactive.

### Simple Elements

**Tags**  
Building blocks like <p> for paragraphs.

**Attributes**  
Extra info for tags.

**Links and Images**  
Connecting pages and adding pictures.

### Analogy  
Web development is like building a house: HTML is the structure, CSS is the paint, JavaScript is the electricity.
