# GitAndGitHub_01_BASICS

## The Alphabet of GitAndGitHub

*Target Audience*: Absolute beginner (Grade school student / Non-technical person).  
*Philosophy*: Assume NO prior knowledge. Explain "what" and "why" before "how".

### Installation & Setup

**What is Git and GitHub?**  
Git is a tool to track changes in files, like a time machine for code. GitHub is a website to share and collaborate.

**Why Git?**  
Keeps history, allows teamwork.

**How to start?**  
Install Git, create GitHub account.

**Fundamental Concepts**  
- **Repo**: Project folder.  
- **Commit**: Save changes.

**Basic Data Types**  
- Files, changes.

**Simple Logic**  
- Add, commit, push.

**Analogy**  
Git is like saving game progress.
