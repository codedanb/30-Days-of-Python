# Async Programming Tutorial

This folder contains the complete async programming tutorial and code examples.

## Contents

- `tutorial.md`: The full tutorial text
- `first_async.py`: Your first async program
- `multiple_tasks.py`: Running multiple async tasks concurrently
- `fastapi_example.py`: FastAPI async web server example
- `async_patterns.py`: Common async patterns and best practices
- `requirements.txt`: Python dependencies

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Run the examples: `python first_async.py`, etc.
3. For FastAPI: `uvicorn fastapi_example:app --reload`

## Tutorial Structure

The tutorial builds concepts from absolute basics to advanced async patterns, following this order:

1. Programming fundamentals
2. The core problem async solves
3. Concurrency vs parallelism
4. Async fundamentals
5. Async mechanics
6. Step-by-step mastery with code examples
7. Teaching enhancements
8. Real-world applications

Read `tutorial.md` for the complete guided learning journey.
