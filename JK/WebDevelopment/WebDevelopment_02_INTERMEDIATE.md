# WebDevelopment_02_INTERMEDIATE

## Building Dynamic Websites

*Target Audience*: Beginner with basic programming knowledge (High school student / Early career).  
*Philosophy*: Build on fundamentals, introduce server-side programming and databases.

### Server-Side Programming

**Backend**  
Code running on the server.

**Frameworks**  
Tools like Flask or Django for Python.

**APIs**  
Ways for front-end and back-end to talk.

### Databases

**What is a Database?**  
Storing data persistently.

**SQL**  
Language for querying databases.

**ORMs**  
Object-Relational Mappers for easier data handling.

### User Interaction

**Forms**  
Collecting user input.

**Sessions**  
Remembering users across pages.

**Authentication**  
Logging in users.

### Templating

**Jinja2**  
Rendering dynamic HTML.

**Variables and Loops**  
Making pages change based on data.

### Next Steps
