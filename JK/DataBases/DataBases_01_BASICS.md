# DataBases_01_BASICS

## The Alphabet of DataBases

*Target Audience*: Absolute beginner (Grade school student / Non-technical person).  
*Philosophy*: Assume NO prior knowledge. Explain "what" and "why" before "how".

### Installation & Setup

**What is a Database?**  
A database is like a digital filing cabinet where you store information organized in tables, rows, and columns.

**Why Databases?**  
To keep data safe, searchable, and updatable. Without them, data is scattered like loose papers.

**How to start?**  
Use free tools like SQLite or online DBs.

**Fundamental Concepts**  
- **Tables**: Like spreadsheets.  
- **Rows**: Records.  
- **Columns**: Fields.

**Basic Data Types**  
- Text, numbers.

**Simple Logic**  
- Query data.

**Analogy**  
Database is like a library catalog: organized books (data) easy to find.
