# AZURE_01_BASICS

## The Alphabet of AZURE

*Target Audience*: Absolute beginner (Grade school student / Non-technical person).  
*Philosophy*: Assume NO prior knowledge. Explain "what" and "why" before "how".

### Installation & Setup

**What is AZURE?**  
Azure is Microsoft's cloud platform, like a huge online workshop where you can build and run apps using virtual machines, storage, and AI tools.

**Why AZURE?**  
It integrates well with Microsoft products. Pay-as-you-go for flexibility.

**How to start?**  
Sign up at azure.com, use portal.

**Fundamental Concepts**  
- **Subscriptions**: Accounts.  
- **Resource Groups**: Containers.  
- **Services**: VMs, Blob storage.

**Basic Data Types**  
- Data blobs.

**Simple Logic**  
- Deploy resources.

**Analogy**  
Azure is like a Lego set in the cloud: build anything with pieces.
