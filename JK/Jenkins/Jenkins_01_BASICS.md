# Jenkins_01_BASICS

## The Alphabet of Jenkins

*Target Audience*: Absolute beginner (Grade school student / Non-technical person).  
*Philosophy*: Assume NO prior knowledge. Explain "what" and "why" before "how".

### Installation & Setup

**What is Jenkins?**  
Jenkins is a tool to automate building and testing software, like a robot chef for code.

**Why Jenkins?**  
Speeds up development by automating tasks.

**How to start?**  
Install Jenkins, create job.

**Fundamental Concepts**  
- **Job**: Task.  
- **Build**: Run job.

**Basic Data Types**  
- Scripts.

**Simple Logic**  
- Trigger build.

**Analogy**  
Jenkins is like an assembly line.
