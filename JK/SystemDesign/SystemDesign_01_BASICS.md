# SystemDesign_01_BASICS

## Introduction to System Design

*Target Audience*: Absolute beginner (Grade school student / Non-technical person).  
*Philosophy*: Assume NO prior knowledge. Explain "what" and "why" before "how".

### What is System Design?

**Systems**  
Collections of parts working together.

**Design**  
Planning how parts interact.

**Why System Design?**  
To build reliable, efficient applications.

### Basic Components

**Clients and Servers**  
Users and computers that serve them.

**Networks**  
Connections between computers.

**Databases**  
Places to store data.

### Simple Concepts

**Scalability**  
Handling more users.

**Reliability**  
Systems that don't break easily.

**Security**  
Protecting data and users.

### Analogy  
System design is like planning a city: roads, buildings, utilities all working together.
