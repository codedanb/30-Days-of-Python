# SQL_02_INTERMEDIATE

## Structuring and Analyzing Data

*Target Audience*: Beginner with basic database knowledge (High school student / Early career).  
*Philosophy*: Build on fundamentals, introduce joins, aggregations, and data manipulation.

### Joins

**INNER JOIN**  
Combining data from multiple tables.

**LEFT/RIGHT JOIN**  
Including all records from one table.

**FULL OUTER JOIN**  
Combining all records.

### Aggregation

**GROUP BY**  
Grouping data for analysis.

**Aggregate Functions**  
SUM, COUNT, AVG, MAX, MIN.

### Subqueries

**Nested Queries**  
Queries within queries.

### Data Modification

**Constraints**  
Ensuring data integrity (PRIMARY KEY, FOREIGN KEY).

**Indexes**  
Speeding up queries.

### Views and Stored Procedures

**Views**  
Virtual tables.

**Stored Procedures**  
Reusable SQL code.
