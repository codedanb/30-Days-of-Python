### Installation & Setup

**What is MCP?**  
MCP is a protocol for AI models to communicate with tools.

**Why MCP?**  
Standardizes AI-tool interactions.

**How to start?**  
Install MCP SDK.

**Fundamental Concepts**  
- **Servers**: Tool providers.  
- **Clients**: AI models.

**Basic Data Types**  
- Messages.

**Simple Logic**  
- Request-response.

**Analogy**  
MCP is like a language for AI and tools.
