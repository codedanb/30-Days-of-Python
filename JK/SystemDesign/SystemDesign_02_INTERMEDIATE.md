# SystemDesign_02_INTERMEDIATE

## Building Scalable Systems

*Target Audience*: Beginner with basic programming knowledge (High school student / Early career).  
*Philosophy*: Build on fundamentals, introduce load balancing, caching, and basic architectures.

### Core Concepts

**Load Balancing**  
Distributing work across servers.

**Caching**  
Storing frequently used data for quick access.

**Databases**  
Choosing between SQL and NoSQL.

### Architecture Patterns

**Monolithic vs Microservices**  
Single application vs small services.

**Client-Server Model**  
Basic request-response.

### Data Management

**Replication**  
Copying data for reliability.

**Sharding**  
Splitting data across servers.

### Communication

**APIs**  
Interfaces for services to talk.

**Message Queues**  
Asynchronous communication.
