# UnitTesting_02_INTERMEDIATE

## Writing Unit Tests

*Target Audience*: Beginner with basic programming knowledge (High school student / Early career).  
*Philosophy*: Build on fundamentals, introduce unit testing frameworks and best practices.

### Unit Tests

**What is a Unit?**  
Smallest testable part of code (function, method).

**Unit Test**  
Test for a single unit.

### Testing Frameworks

**unittest**  
Python's built-in testing framework.

**pytest**  
Popular alternative with simpler syntax.

### Test Structure

**Setup**  
Preparing test data.

**Execution**  
Running the code to test.

**Assertion**  
Checking the results.

### Common Assertions

**assertEqual**  
Check if two values are equal.

**assertTrue/False**  
Check boolean conditions.

**assertRaises**  
Check if exceptions are raised.

### Test Organization

**Test Classes**  
Grouping related tests.

**Test Methods**  
Individual test functions.
