# DevOps_01_BASICS

## The Alphabet of DevOps

*Target Audience*: Absolute beginner (Grade school student / Non-technical person).  
*Philosophy*: Assume NO prior knowledge. Explain "what" and "why" before "how".

### Installation & Setup

**What is DevOps?**  
DevOps is a way to make building and running software faster and better by combining development and operations teams.

**Why DevOps?**  
It reduces errors and speeds up delivery, like a well-oiled machine.

**How to start?**  
Learn basic tools like Git.

**Fundamental Concepts**  
- **CI**: Continuous Integration.  
- **CD**: Continuous Delivery.

**Basic Data Types**  
- Code, configs.

**Simple Logic**  
- Build, test, deploy.

**Analogy**  
DevOps is like a factory assembly line for software.
