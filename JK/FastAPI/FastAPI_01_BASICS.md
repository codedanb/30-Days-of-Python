# FastAPI_01_BASICS

## The Alphabet of FastAPI

*Target Audience*: Absolute beginner (Grade school student / Non-technical person).  
*Philosophy*: Assume NO prior knowledge. Explain "what" and "why" before "how".

### Installation & Setup

**What is FastAPI?**  
FastAPI is a tool to build web applications quickly, like a fast car for creating APIs.

**Why FastAPI?**  
It's fast and easy, with automatic docs.

**How to start?**  
Install with pip, create app.py.

**Fundamental Concepts**  
- **Routes**: Paths for requests.  
- **Requests**: What users send.

**Basic Data Types**  
- JSON data.

**Simple Logic**  
- Handle GET/POST.

**Analogy**  
FastAPI is like a fast food drive-thru for web requests.
